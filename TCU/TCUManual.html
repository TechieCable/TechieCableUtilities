TechieCableUtilities

TCUManual

TechieCableUtilities is a lightweight, independent program made by TechieCable. It includes some of TechieCable's favorite utilities - all in one small program! TCU currently includes:

  • AlwaysOnTop - pin programs "on top" of all other programs!
  • TouchPad - turn touchpad on/off
  • SpecChars - type some Unicode characters more easily

TCU comes with some simple options for its users.

Note: stars * in the brackets [] represent the level of difficulty, with five [*****] being most difficult and one [*] being the least difficult

Note: The parentheses () note what the instructions apply to

Turn features on/off [*] (General)

  • Find TechieCableUtilities in your taskbar
  • Right-click TechieCableUtilities
  • Enter the "Options (Hotkeys)" menu
  • Left-click features on/off

**This feature disables/enables the HOTKEYS (mentioned below)

Pin/Unpin a program "on top" [**] (AlwaysOnTop)

  • Pin a program
    • Select the program you want to put "on top"
    • Press CTRL + ALT + SHIFT + Space
  • Unpin a program
    • Select a pinned program
    • Press CTRL + ALT + SHIFT + SPACE

To activate/deactivate the touchpad [*] (TouchPad)

  • Press CTRL + F2

OR

  • Find TechieCableUtilities in your taskbar
  • Right-click TechieCableUtilities
  • Left-click "TouchPadToggle"

** TouchPad only works on some computers and requires a touchpad to be included in the computer. Please check if your computer has a touchpad before using

** TouchPad relies on an external script (data\TouchpadToggle.exe) to work. You may have to allow this program through your security/antivirus program

Use the AlwaysOnTop Menu [**] (AlwaysOnTop)

  • Press CTRL + ALT + SHIFT + UP
  • Select a program from the list
  • Do the same to unpin a program

** You may have to activate the program (click it) for it to recognize that it is no longer "on top"

Disable a feature [***] (General)

  • Add the following to TCU.ini
    [disabled]
  • And then choose one or more of the following:
    disable_AOT=1
disable_TouchPad=1
disable_loading=1
  • Remove any of the above to allow the feature again (or set =0 instead)

** Use disable_loading if you don't want to see the TCULauncher loading screen

Delete TechieCableUtilities [*] (General)

If you don't want TechieCableUtilities anymore, something is broken, or you want to uninstall to reinstall, you can remove TechieCableUtilites from your computer. To do this, you can either delete TechieCableUtilities' folder or use the built-in removal function

  • OPTION ONE:
    • Right-click TechieCableUtilities in your toolbar and then click "Open Script Directory"
    • Right-click TechieCableUtilities again and click "Exit"
    • In the window that opened, go up a folder and delete the folder called "TechieCableUtilities"
  • OPTION TWO
    • Right-click TechieCableUtilties in your toolbar and click "Remove TCU" (Under the TechieCableUtilites menu)
    • A small window may briefly appear; this is the command deleting the folder
    • Done!

Customization

Use a custom icon for the options menu [***] (General)

  • Replace data\settings_cog.ico with your own .ico file
  • Reload the program

** To revert to the original, simply delete data\settings_cog.ico and restart the program

Use custom hotkeys: Part 1 [**] (General)

  • In v1.0.8, customizing hotkeys became so much easier!
  • Right-click TechieCableUtilities and click "Hotkey Customizer"
  • In each box, type the hotkey you want to use
  • When you finish, click "Submit"

** Some keys may not work - if you really want to use one of these, follow the directions below

Use custom hotkeys - Part 2 [****] (General)

  • These instructions are more advanced: refer to Part 1 unless you have to use this method
  • Add the following to TCU.ini
    [hotkey]
  • And then specify the keyboard shortcut after the following
    AOT_key=_____
AOTMenu_key=_____
TouchPad_key=_____
  • Visit the AutoHotkey docs website for instructions on writing hotkeys

Add Special Characters [*****] (SpecChars)

  • Create a file in the TCU directory called SpecChars.txt
  • Add :*:, followed by the characters that are replaced
  • Then add :: followed by the Unicode ID for the character you want to appear in {}

For example, :*:`()::{U+00B0} replaces `() with {U+00B0}, which is °

  • Try placing :*:`()::{U+00B0} in your SpecChars.txt file to test it out!

HINT: Search for "Unicode " followed by the character you want on your internet browser.

Addons [*****] (General)

__This is a very advanced feature. Make sure to read the resources found at the AutoHotkey docs website

  • TechieCableUtilities searches its data directory for a file called addon.txt and runs anything in that file
  • If you need to add a callable function, put it in a file called gosub.txt (in the data directory) and call it with "gosub" or a timer in the addon.txt file
  • If you are feeling really adventurous, try adding code into this file (in AutoHotkey)!
  • TechieCableUtilities is written in AutoHotkey, which you can learn more about at www.autohotkey.com
